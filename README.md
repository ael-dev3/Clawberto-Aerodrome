# Clawberto Aerodrome

Production dashboard for Aerodrome CL200 LFI/USDC on Base, comparing Clawberto's AI wallet against the manual wallet.

https://ael-dev3.github.io/Clawberto-Aerodrome/

- Live Base RPC attribution for wallet-held and gauge-staked Slipstream NFTs.
- Separate AI and human price charts with wallet-specific LP range overlays.
- Profitability index, range uptime/downtime, APR, pending emissions, and LP side split per wallet.
- LP range-width model using GeckoTerminal candles, weekday/hour volatility heatmap, and emissions APR.
