# Clawberto Aerodrome

Production dashboard for Aerodrome CL200 LFI/USDC on Base, comparing Clawberto's AI wallet against the manual wallet.

https://ael-dev3.github.io/Clawberto-Aerodrome/

- Live Base RPC attribution for wallet-held, gauge-staked, and linked LP-controller NFTs.
- Separate AI and human price charts with wallet-specific LP range overlays.
- Tier score, two-state range uptime, overall PnL, APR, pending emissions, pool share, and LP side split per wallet.
- LP width model using GeckoTerminal history, weekday/hour volatility heatmap, and live Aerodrome reward density.
- Diagnostics include the audited LP position history that release sync appends after each dashboard-safe rebalance update.
